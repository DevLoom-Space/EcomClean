This folder stores image assets for the DevLoom app.

Guidelines:
- Put image files here (png, jpg, webp, svg).
- Use descriptive, kebab-case filenames (e.g. product-acer-nitro-16.png).
- For templates, reference images using the `{% static %}` tag: `{% static 'devloom/images/your-file.png' %}`.
- Keep images < 200 KB when possible for faster load times. Use WebP for best compression.

If you want, I can add an example image or a small script to optimize images automatically.
